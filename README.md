# MoodPad

A mood-aware macropad with GRIND and CHILL modes.

## Features

- Seeeduino XIAO microcontroller
- 0.91" OLED display
- Rotary encoder with push button
- 8-key matrix
- Custom PCB designed in KiCad
- Custom enclosure designed in Tinkercad
- Dual operating modes

## Modes

### ⚡ GRIND

- ChatGPT
- GitHub
- KiCad
- Notion
- Google Drive
- Google Docs
- Screenshot
- Focus Timer

### 🌙 CHILL

- Spotify
- YouTube
- Discord
- Netflix
- Play/Pause
- Next Track
- Previous Track
- Mute

## Hardware

- Seeeduino XIAO
- 0.91" I2C OLED
- Rotary Encoder
- 8 Mechanical Switches

## Project Files

- PCB files in `/PCB`
- Firmware in `/Firmware`
- Enclosure files in `/Case`
- Screenshots in `/Images`

## Development

MoodPad was designed using KiCad for electronics and Tinkercad for the enclosure. The project combines PCB design, firmware architecture, and product design into a compact desktop macropad.
